# 🤖 Lenu — Rule-Based AI Chatbot
**DecodeLabs AI Internship | Batch 2026 | Project 1**

## What is this?
Lenu is a friendly rule-based chatbot built using pure Python control flow.
No machine learning — just clean logic,dictionaries,and decision-making.

## Key Concepts Applied
- O(1) dictionary lookup instead of slow if-elif chains
- Input sanitization using .lower().strip()
- Infinite while loop with a clean exit command
- Fallback response for unknown inputs
- IPO model: Input→Process→Output

## Sample Conversation
You: hello
Lenu: Hello there! I'm Lenu, what can I help you with today?
You: motivate me
Lenu: Small progress every day becomes huge success.
You: exit
Lenu: Goodbye! Session ended.

## Main Takeaway
Rule-based logic is the skeleton every AI system is built on —
before a machine can think, it must learn to obey.
