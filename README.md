## Project 2: Data Classification Using AI

### Description
This project uses the Iris Dataset and the K-Nearest Neighbors (KNN) algorithm to classify flowers into different species.

### Dataset
- Iris Dataset
- 150 samples
- 3 classes:
  - Setosa
  - Versicolor
  - Virginica

### Technologies Used
- Python
- NumPy
- Matplotlib
- Scikit-Learn

### Concepts Implemented
- Exploratory Data Analysis (EDA)
- Feature Scaling using StandardScaler
- Train-Test Split
- KNN Classification
- Confusion Matrix
- Classification Report
- K Value Analysis
- Model Comparison (KNN vs Decision Tree)

### Features
- Finds the best K value
- Compares multiple machine learning models
- Predicts species for custom flower measurements
- Visualizes model performance using graphs

### Learning Outcomes
Through this project,I learned:
- Supervised Learning
- Data Preprocessing
- Feature Scaling
- Model Evaluation
- Classification Techniques
